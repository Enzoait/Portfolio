# Portfolio — Enzo Ait-Yakoub

## 🇫🇷 Français

Portfolio professionnel

http://enzoaityakoub.com 

**Description**

- Portfolio personnel construit avec Next.js 15, React 19 et Tailwind CSS 4. Le site est multilingue (FR/EN), inclut un formulaire de contact avec envoi d’e-mails, un mode clair/sombre, des animations fluides et des composants UI accessibles.

**Technologies utilisées**

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4 + @tailwindcss/postcss
- Radix UI & Headless UI (primitives et composants accessibles)
- next-themes (gestion du thème clair/sombre)
- motion (animations)
- lucide-react & heroicons (icônes)
- sonner (toasts/notifications)
- nodemailer + googleapis (envoi d’emails côté serveur)
- negotiator + @formatjs/intl-localematcher (détection locale pour i18n)
- tailwind-merge (utilitaires CSS)

## 🇬🇧 English

**Title**

Professionnal portfolio

http://enzoaityakoub.com 

**Description**

- Personal portfolio built with Next.js 15, React 19, and Tailwind CSS 4. The site is bilingual (EN/FR), includes a contact form with email sending, light/dark theme, smooth animations, and accessible UI components.

**Technologies Used**

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4 + @tailwindcss/postcss
- Radix UI & Headless UI (accessible primitives/components)
- next-themes (light/dark theming)
- motion (animations)
- lucide-react & heroicons (icons)
- sonner (toasts/notifications)
- nodemailer + googleapis (server-side email)
- negotiator + @formatjs/intl-localematcher (locale detection for i18n)
- tailwind-merge (CSS utilities)
